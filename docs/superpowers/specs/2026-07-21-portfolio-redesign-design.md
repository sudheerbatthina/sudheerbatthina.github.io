# AI Engineering Portfolio Redesign

## Goal

Position Sudheer Batthina as a production-minded AI/ML engineer whose work moves beyond demos into reliable, measurable systems.

## Direction

- Static GitHub Pages site with no framework or build dependency.
- “Systems Blueprint” structure with a restrained Signal Coral palette.
- Recruiter scan order: positioning, proof, flagship work, capabilities, experience, credentials, contact.
- Use distinctive professional typography, strong contrast, responsive layouts, and purposeful motion with reduced-motion support.
- Feature Sourcebound, RemaindALL, and StyleTrip.ai. Keep private product links non-clickable unless a public destination exists.
- Retain Job Matcher and Job Autofill as public engineering evidence.
- Remove the public phone number, empty projects, badge-wall presentation, inflated seniority language, and unverified “live” labels.

## Copy Position

Primary headline: “I build AI systems that move from prototype to dependable production.”

Supporting copy: “AI/ML engineer specializing in agentic workflows, RAG, evaluation, and scalable ML systems—engineered around reliability, security, and measurable outcomes.”

## Quality Bar

- Semantic HTML and keyboard-visible focus states.
- WCAG-friendly color contrast and tap targets.
- Mobile navigation without JavaScript dependency.
- Fast-loading static assets and no decorative canvas animation.
- Accurate product names, including exact spelling “RemaindALL.”
